# CLIタスク管理ツール「my-todo」

## 概要

ターミナルから操作するタスク管理ツールです。<br>
タスクの登録、一覧確認、ステータス更新、削除、検索、統計表示が可能です。

## セットアップ手順

### パッケージのインストール

下記のコマンドでパッケージをインストールしてください。

```
npm install
```

## 各コマンドの使い方

### add コマンド

タスクを追加します。

```
node todo.js add "牛乳を買う"
```

成功した場合、メッセージが緑色で表示されます。

```
タスクを追加しました. ID: e1b20820-99ab-457a-a9ab-5739f858ee0d, タイトル: 牛乳を買う, 作成日時: 2026-04-19 20:41, 完了状態: 未完了, 優先度: medium
```

※各タスクには一意のIDが付与されます。<br>
※作成日時は `YYYY-MM-DD HH:mm` 形式で保存されます。

#### 優先度について

優先度は `high` / `medium` / `low` の3段階です。<br>
`--priority` を指定することで、任意の優先度を設定することができます。<br>
`--priority` を省略した場合は、デフォルトで `medium` が設定されます。

```
node todo.js add "卵を買う" --priority high
node todo.js add "チーズを買う" --priority low
```

### list コマンド

登録タスクの一覧を表示します。

```
node todo.js list
```

成功した場合、登録されているタスクが一覧で表示されます。

```
ID: e1b20820-99ab-457a-a9ab-5739f858ee0d, タイトル: 牛乳を買う, 作成日時: 2026-04-19 20:41, 完了状態: 完了, 優先度: medium
ID: e62addeb-5598-4f02-bf68-16866bb57ec4, タイトル: 卵を買う, 作成日時: 2026-04-19 20:42, 完了状態: 未完了, 優先度: high
ID: 2b09b738-e605-4adb-a051-89ecf6e6aa22, タイトル: チーズを買う, 作成日時: 2026-04-19 20:43, 完了状態: 未完了, 優先度: medium
ID: 5041be70-4cb6-451d-b3a0-8b7724ad8b35, タイトル: パンを買う, 作成日時: 2026-04-19 20:44, 完了状態: 未完了, 優先度: low
```

※タスク毎に ID、タイトル、作成日時、完了状態、優先度 が表示されます。<br>
※完了状態が完了の場合、灰色で表示されます。<br>
※完了状態が未完了の場合、優先度ごとに色分けされます。<br>
　「high → 赤色」　「medium → 黄色」　「low → 白色」

#### フィルタ表示について

`--done` を指定することで、完了タスクのみを表示することができます。<br>
`--todo` を指定することで、未完了タスクのみを表示することができます。

```
node todo.js list --done
node todo.js list --todo
```

### done コマンド

指定タスクを完了状態にします。

```
node todo.js done <id>
```

成功した場合、メッセージが緑色で表示されます。

```
タスクを完了にしました. ID: e1b20820-99ab-457a-a9ab-5739f858ee0d, タイトル: 牛乳を買う, 作成日時: 2026-04-19 20:41, 完了状態: 完了, 優先度: medium
```

### delete コマンド

指定タスクを削除します。

```
node todo.js delete <id>
```

成功した場合、メッセージが黄色で表示されます。

```
タスクを削除しました. ID: e1b20820-99ab-457a-a9ab-5739f858ee0d, タイトル: 牛乳を買う, 作成日時: 2026-04-19 20:41, 完了状態: 完了, 優先度: medium
```

### search コマンド

タスクのタイトルに対して部分一致検索を行います。

```
node todo.js search "牛乳"
```

成功した場合、該当するタスクが表示されます。

```
ID: e1b20820-99ab-457a-a9ab-5739f858ee0d, タイトル: 牛乳を買う, 作成日時: 2026-04-19 20:41, 完了状態: 完了, 優先度: medium
```

※表示形式は `list` コマンドと同じです。

### stats コマンド

登録されているタスクの統計を表示します。

```
node todo.js stats
```

成功した場合、以下の情報が表示されます。

- 全タスク数
- 完了タスク数
- 未完了タスク数
- 完了率（パーセンテージ）
- 直近7日以内に作成されたタスクの件数

```
全タスク数: 9, 完了タスク数: 3, 未完了タスク数: 6, 完了率: 33.3%, 直近7日以内に作成されたタスクの件数: 8
```
