# CLIタスク管理ツール「my-todo」

## 概要

ターミナルから操作するタスク管理ツールです。<br>
タスクの登録、一覧確認、ステータス更新、削除が可能です。

## セットアップ手順

### パッケージのインストール

下記のコマンドでパッケージをインストールしてください。

```
npm install
```

## 各コマンドの使い方

### add コマンド

タスクを追加します。

```
node todo.js add "牛乳を買う"
```

成功した場合、メッセージが緑色で表示されます。

```
タスクを追加しました. ID: d0468286-62c9-473d-beaf-42b6fab5e080, タイトル: 牛乳を買う
```

※各タスクには一意のIDが付与されます。<br>
※作成日時は `YYYY-MM-DD HH:mm` 形式で保存されます。

### list コマンド

登録タスクの一覧を表示します。

```
node todo.js list
```

成功した場合、登録されているタスクが一覧で表示されます。

```
ID: d0468286-62c9-473d-beaf-42b6fab5e080, タイトル: 牛乳を買う, 作成日時: 2026-03-26 14:06, 完了状態: 未完了
ID: d2722027-4b43-4b30-b131-efbff879112a, タイトル: 卵を買う, 作成日時: 2026-03-26 14:12, 完了状態: 完了
```

※タスク毎に ID、タイトル、作成日時、完了状態 が表示されます。<br>
※完了状態が未完了の場合、白色で表示されます。<br>
※完了状態が完了の場合、灰色で表示されます。

### done コマンド

指定タスクを完了状態にします。

```
node todo.js done <id>
```

成功した場合、メッセージが緑色で表示されます。

```
タスクを完了にしました. ID: d2722027-4b43-4b30-b131-efbff879112a, タイトル: 卵を買う
```

### delete コマンド

指定タスクを削除します。

```
node todo.js delete <id>
```

成功した場合、メッセージが緑色で表示されます。

```
タスクを削除しました. ID: d2722027-4b43-4b30-b131-efbff879112a, タイトル: 卵を買う
```

## 使用したライブラリの一覧

| ライブラリ | 用途 |
| ------------- | -------------------------------------------- |
| *commander* | コマンドライン引数の解析（サブコマンド定義） |
| *chalk* | ターミナル出力の色付け |
| *dayjs* | 日時のフォーマット |
| *uuid* | 一意なIDの生成 |
